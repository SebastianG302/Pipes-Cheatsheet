Basic Pipes

Pipes that are typically included in Common Module.

e.g. "{{name}}"

name | lowercase
  • {{name | lowercase}}
name | uppercase
  • {{name | uppercase}}
name | titleclase
  • {{name | titlecase}}

Date Pipes

Pipes that are typically included in Common Module.

e.g. "{{todaysDate}}"

todaysDate | date
todaysDate | date:'short'
todaysDate | date:'long'
todaysDate | date:'EEEE, yyyy, G' //custom pipe
todaysDate | date:'long': 'GMT:-6'
todaysDate | date:'long': 'GMT:-6': 'es'
  • {{ todaysDate | date }}
  • {{ todaysDate | date:'short' }}
  • {{ todaysDate | date:'long' }}
  • {{ todaysDate | date:'EEEE, yyyy, G' }}

  • {{ todaysDate | date:'long': 'GMT:-6' }}
  • {{ todaysDate | date:'long': 'GMT:-6': 'es' }}
